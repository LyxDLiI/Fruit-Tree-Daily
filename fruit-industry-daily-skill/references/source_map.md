# 来源与关键词参考

## A. 建议优先来源

### 官方/公共机构（优先级最高）
- 农业农村部（政策、全国市场监测）：https://www.moa.gov.cn/
- 全国农产品批发市场价格信息系统（批发价格）：http://pfsc.agri.cn/
- 中国农业信息网（生产与行情）：http://www.agri.cn/
- 中国气象局及地方气象台（灾害天气）：https://www.cma.gov.cn/
- 海关总署（进出口）：http://www.customs.gov.cn/
- 各省农业农村厅（区域政策与技术通知）：对应省级官网链接（示例：广东 https://dara.gd.gov.cn/）

### 行业与科研
- 国家现代农业产业技术体系果树岗位：http://www.caas.cn/
- 各省农科院、果树研究所：优先机构官网（示例：中农院郑果所 http://zzgss.caas.cn/）
- 农技推广站/技术培训公告：优先省市农业农村部门官网公告页

## A2. URL 采集规范
- 必须记录“原文页面 URL”，不能只写站点首页。
- 同一信息至少保留 1 个官方 URL；如有冲突，以官方口径为准。
- 避免短链、跳转聚合页，优先最终落地页链接。
- 在日报中显示可点击 Markdown 链接，格式：`[来源名-标题](URL)`。

## B. 季节化关键词（可按月替换）
- 1-2月：清园、修剪、越冬防寒、设施管理
- 3-4月：萌芽、花期授粉、倒春寒、防冻
- 5-6月：坐果、疏果、病虫高发、夏剪
- 7-8月：膨果、裂果防控、日灼、排涝
- 9-10月：采收、分级、采后保鲜、冷链
- 11-12月：采后修复、基肥、冬剪、土壤改良

## C. 主要品类扩展词
- 苹果：花期、幼果期、套袋、轮纹病、炭疽叶枯病
- 柑橘：保花保果、黄龙病、溃疡病、红蜘蛛
- 葡萄：花序管理、灰霉病、霜霉病、裂果
- 香蕉：枯萎病、寒害、采后催熟、品质分级
- 西瓜：授粉坐果、蔓枯病、炭疽病、棚温管理
